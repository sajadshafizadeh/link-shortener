## Installation:
You need to run the following command when you're under root of the project. 

    composer install

----

## Run:
You need to run the following command when you're under root of the project. 

    php -S localhost:8000 -t web

Then the website is accesible here: `http://localhost:8000/` 

Sample URL: http://localhost:8000/hello/Sajad
